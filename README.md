# ⚡ PowerSupply System

A modern smart energy platform built with **Flask**, **PostgreSQL**, and a fully interactive frontend.  
The system combines an **energy marketplace**, **AI assistance**, **admin management**, and **user dashboards** into one centralized platform.

---

# 🌍 Overview

PowerSupply System is a full-stack energy management and marketplace platform designed to simulate a modern renewable energy ecosystem.

The platform currently works fully on the frontend using **Local Storage**, while the backend migration to **Flask + PostgreSQL** is actively in development.
